# 대호에게 — 좀비 웨이브 디펜스 업데이트 소식

안녕 대호야! 네가 보내준 피드백 4개를 보고 게임을 고쳤어.
먼저 이 말부터 할게. **네가 문제를 번호 붙여서 정리해준 거, 진짜 게임 회사 테스터들이 하는 방식 그대로야.** 덕분에 빠르게 고칠 수 있었어. 고마워!

## 네가 찾은 문제, 이렇게 고쳤어

### 1. "맵이 안 보여요"
네 말이 맞았어. 일부 아이폰에서 맵의 키(높이)가 0이 되어버리는 문제였어.
이제 맵이 항상 일정 크기 이상으로 보이게 고쳤어.

### 2. "구매를 누르면 이상한 화면으로 가요"
이것도 진짜 버그였어. 보상 팝업창이 화면 가운데에 떠야 하는데,
일부 폰에서는 페이지 맨 아래에 길게 펼쳐져서 "다른 화면"처럼 보였던 거야.
이제 어떤 폰에서도 가운데에 예쁘게 뜨도록 고쳤어.

### 3. "카드 설명이 잘려요"
설명 글자를 2줄까지만 보여주고 자르는 코드가 숨어 있었어. 지웠어.
이제 설명이 끝까지 다 보여.

### 4. "랭킹에 이상한 게 나와요"
사실 그건 게임을 만들 때 연습용으로 넣어둔 가짜 데이터였어. 들킨 거지!
이제 랭킹에는 **너의 진짜 최고 기록**(최고 생존 일수, 최고 웨이브, 플레이 횟수)이 나와.
기록을 깨려면 더 오래 살아남아야 해.

## 보너스로 고친 것

- 캐릭터 그림이 안 보이던 문제도 고쳤어 (그림이 게임 파일 안에 들어있지 않았거든)
- 이제 보석, 카드, 강화한 것들이 **자동 저장**돼. 게임을 껐다 켜도 안 사라져!

## 우리가 테스트하다가 발견한 것

좋은 소식: **이제 백신 엔딩을 진짜로 깰 수 있어!**
(힌트: 연구소에서 조사를 여러 번 해서 데이터를 모으고, 연구를 반복하면 돼)

고민할 것: 열심히 플레이해도 6일차쯤 기지가 무너져. 왜냐하면 지금은
**공격 버튼을 눌러도 밤에 오는 좀비 수가 줄어들지 않거든.**

## 대호 디자이너에게 질문!

다음 업데이트를 위해 네 생각이 필요해.

1. 낮에 좀비를 공격하면, 그만큼 밤에 오는 좀비가 줄어들게 바꿀까?
2. 처음 시작하는 사람에게 직업 카드 1장을 공짜로 줄까? (지금은 다 시민으로 시작해)
3. 게임이 좀 어려운 것 같은데, 며칠까지 버티면 적당하다고 생각해?

플레이해 보고 또 피드백 보내줘. 네 아이디어가 이 게임을 만들고 있어!
