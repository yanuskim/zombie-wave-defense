## 1. 프로젝트 개요

**좀비 웨이브 디펜스**는 턴제 생존 전략 게임이다. 좀비 바이러스가 창궐한 2089년 도시를 배경으로, 플레이어는 기지를 지키면서 단서를 수집하고 생존 조건을 갖춰 다양한 엔딩에 도달한다.

**핵심 컨셉**
- 마피아42의 팀 협력·역할 분배 구조에서 영감
- 직업 카드 + 티어 능력 강화로 플레이 스타일 차별화
- 제한된 행동력(6개/턴)으로 매 턴 우선순위 결정
- 6가지 엔딩 분기 (조건 충족 시 가장 높은 등급 엔딩 발동)

**세계관 요약**
- 배경: 2089년 엔터톤사이언스가 죠나바이러스를 비밀리에 방사
- 조직: 데드저스티스 — 엔터톤사이언스의 음모를 막기 위해 창립된 반(反)기업 단체
- 플레이어: 서바이벌스 팀 (의사·사냥꾼·군인·시민)

## 3. 게임 흐름

```
타이틀 화면
    ↓ [터치해서 시작]
로비 (직업 카드 강화 / 유물 선택 / 상점 / 업적 등)
    ↓ [작전 시작]
준비 화면 (작전 준비 — 직업·유물 최종 선택)
    ↓ [작전 시작]
1일차 오전 7시 — 토의 페이즈 (90초 / 건너뛰기 가능)
    ↓
오전 9시 — 게임 페이즈 시작
    ↓
[행동 소비: 이동/공격/제작/치료/연구/조사/탐험 중 선택]
    ↓
[턴 종료] → 식량/물 소비 → 웨이브 공격 (2일차부터)
    ↓
다음 날 토의 페이즈
    ↓
[엔딩 조건 충족 또는 기지/HP 붕괴]
    ↓
엔딩 화면
```

**시간 표시 (행동 소비량 기준)**

| 소비한 행동 수 | 시각 표시 |
|:---:|:---:|
| 0 | 오전 9시 |
| 1 | 오전 11시 |
| 2 | 오후 1시 |
| 3 | 오후 3시 |
| 4 | 오후 5시 |
| 5 | 오후 7시 |
| 6 | 오후 9시 |

## 4. 직업 카드 시스템

### 기본 스탯

| 직업 | 체력 | 공격력 | 보호력 | 고유 능력 |
|------|:---:|:---:|:---:|------|
| 의사 | 100 | 5 | 0 | 전문: 치료량 +10, 타인 치료 가능 |
| 사냥꾼 | 250 | 20 | 3 | 유능: 탐험 시 동물 자원 보장 |
| 군인 | 200 | 15 | 10 | 방탄: 높은 보호력 |
| 시민 | 140 | 5 | 1 | 없음 |

> 초기 보유: 시민 카드만 소유. 의사·사냥꾼·군인은 상점에서 직업 카드 구매 필요.

### 티어 능력 강화

카드 강화로 순서대로 개방 (보석 소비, 카드 1장 = 1단계 강화).

| 직업 | 1티어 | 2티어 | 3티어 | 4티어 |
|------|------|------|------|------|
| 의사 | 진정 | 강화제 | 검진 | 박애 |
| 사냥꾼 | 은탄 | 이타적 유전자 | 항체 보유자 | 추적 |
| 군인 | 불침번 | 불굴 | 정신력 | 사격술 |
| 시민 | — | — | — | — |

### 티어 능력 효과 (v2.0 기준 — 전체 구현 완료)

| 능력 | 효과 | 구현 위치 |
|------|------|------|
| 진정 | 치료 시 대상 HP +3 추가 회복 | heal() |
| 강화제 | 치료 시 대상에게 다음 공격력 +8 버프 부여 (1회 소모) | heal() / attack() |
| 검진 | 의사가 있으면 탐험 사망 판정 -4 | expedition() |
| 박애 | 치료량 +5 | heal() |
| 은탄 | 공격 피해 +4 | attack() |
| 이타적 유전자 | 체력 최대치 +30 (게임 시작 시 적용) | createPlayer() |
| 항체 보유자 | 탐험 사망 판정 -10 | expedition() |
| 추적 | 탐험 행동력 소모 4→3 | expedition() |
| 불침번 | 행동력 +1 (7개/턴) | actionLimit() |
| 불굴 | 보호력 +5 | createPlayer() |
| 정신력 | 팀에 군인이 있으면 야간 육상좀비 피해 40% 감소 | endTurn() |
| 사격술 | 공격력 +10 | createPlayer() |

## 5. 행동 시스템

### 행동력

- 기본 행동력: **6개/턴**
- 군인 `불침번` 강화 시: **7개/턴**

### 7가지 행동과 비용

| 행동 | 소모 | 설명 |
|------|:---:|------|
| 이동(탐색) | 2 | 지도에서 다른 장소로 이동 |
| 공격 | 1 | 좀비 제압 → 기지 압박 완화, 기지 HP 소량 회복 |
| 제작 | 1 | 인벤토리 재료로 아이템 제작 (연구 선행 필요) |
| 치료 | 2 | 가장 부상이 심한 팀원 치료 (의약품 있으면 +10) |
| 연구 | 1 | 제작법 습득 → 이후 백신 연구 (데이터 2 + 의약품 1) |
| 조사 | 1 | 현재 장소 단서 수집 + 건물 이벤트 발동 |
| 탐험 | 4 (추적 시 3) | 위험 구역 돌파 → 자원 획득 or 피해 (생존 판정) |
| 상호작용 | 1 | 장소별 수확: 숲(나무·거미줄), 공사장/폐도시(돌·흙·고철), 마트(식량·물) |

### 탈수·배고픔 게이지

- 탈수·배고픔: 각 0~100 (초기값 100)
- 턴 종료 시 플레이어 수만큼 식량·물 소비
- 부족 시 **모든 플레이어 HP -12**

## 6. 자원 시스템

### 공유 자원 (팀 전체)

| 자원 | 초기값 | 용도 |
|------|:---:|------|
| 식량 | 6 | 턴 소비 (인원 수만큼), 최강의 기지 엔딩 조건 |
| 물 | 6 | 턴 소비 (인원 수만큼), 최강의 기지 엔딩 조건 |
| 고철 | 3 | 무기 제작 (3개 → 강화 무기 1), 달 탈출 연구 재료 |
| 의약품 | 2 | 치료 보너스 (+10), 백신 연구 재료 |
| 데이터 | 0 | 백신 연구 재료 (2개 소비), 달 탈출 조건 |
| 무기 | 0 | 공격력 보조 (+4/개), 탐험 위험도 감소 (-6), 방어 보조 |

### 개인 자원 (플레이어별)

| 항목 | 설명 |
|------|------|
| HP | 직업별 최대 HP (의사 100 / 사냥꾼 250 / 군인 200 / 시민 140) |
| 행동력 | 턴 시작 시 6 (불침번 7) 복구 |
| boosted | 강화제 효과 여부 (다음 공격력 +8, 1회 소모) |

### 진행 자원 (게임 전체)

| 항목 | 설명 |
|------|------|
| 백신 진행도 | 0~100%, 연구 1회당 +25% |
| 데드저스티스 신뢰도 | 0~10, 협력 엔딩 조건 |
| 기지 내구도 | 0~8 (최대 8), 웨이브마다 감소 |
| moonPlan | 달 탈출 정보 입수 여부 (은밀한 상점 조사/이벤트로 획득) |
| forestUnlocked | 숲 지역 잠금 해제 여부 |

## 7. 장소 및 이벤트

### 8개 장소

| 장소 | 위협도 | 특징 |
|------|:---:|------|
| 기지 | 1 | 방어 대상. 이벤트: 비상 식량 or 방벽 보강 |
| 연구소 | 3 | 단서 3개 (lab1~3), 데이터 획득. 이벤트: 데이터+의약품 or 보안 피해 |
| 폐병원 | 2 | 단서 1개 (hospital1), 신뢰도 +3. 이벤트: 의약품 or 신뢰도 +1 |
| 공사장 | 3 | 고철 획득에 유리. 이벤트: 고철 3 or 방벽 보강 +10 |
| 은밀한 상점 | 3 | 단서 1개 (shop1), 달 탈출 정보. 이벤트: 의약품+데이터 or moonPlan 활성 |
| 폐도시 | 4 | 고철 집중 획득. 이벤트: 고철 3 or 건물 붕괴 피해 -10 |
| 마트 | 2 | 식량·물 보급. 이벤트: 식량+물 각 3 or 의약품 1 |
| 숲 (잠금) | 5 | 초기 잠금. 폐도시·마트·은밀한 상점 탐험 성공 시 해제. 이벤트: 식량+물 or 부상 -12 |

### 숲 잠금 해제 조건

폐도시·마트·은밀한 상점 중 어느 한 곳에서 **탐험 성공** 시 자동 해제.

## 8. 제작 시스템

### 사전 조건

**연구 행동** 1회 실행 → `researchedCraft = true` 후 제작 가능.

### 제작표 전체

| 결과물 | 재료 | 언락 조건 |
|------|------|------|
| 땔감 | 나무 3 | 나무 얻기 |
| 막대기 | 나무 2 | 나무 얻기 |
| 목재 | 나무 1 | 나무 얻기 |
| 절구통 | 목재 3 | 목재 얻기 |
| 절구 | 목재 2 | 목재 얻기 |
| 삽 | 막대기 1 + 목재 1 | 목재 얻기 |
| 실 | 거미줄 1 → 실 2 | 숲 진입 후 거미줄 획득 |
| 줄 | 실 4 → 줄 2 | 숲 진입 후 거미줄 획득 |
| 창 | 줄 2 + 돌 1 | 아무 좀비 한 마리 잡기 |
| 텃밭 | 흙 3 | 삽 제작하기 |
| 물레방아 | 나무 15 | 마트 탐험 성공 |
| 조잡한 요리대 | 목재 3 | 목재 제작하기 |

### 재료 획득 경로

| 재료 | 획득 방법 |
|------|------|
| 나무 | 숲 상호작용 (3개), 기타 장소 상호작용 (1개) |
| 거미줄 | 숲 상호작용 (1개) |
| 돌 | 공사장·폐도시 상호작용 (1개) |
| 흙 | 공사장·폐도시 상호작용 (2개) |

### 즉시 제작 (재료 없을 때 폴백)

- 고철 3개 → 강화 무기 1
- 식량 2 + 물 2 → 의약품 1

## 9. 좀비 & 웨이브

### 좀비 타입

| 종류 | 공격력 | HP | 이동속도 | 특징 |
|------|:---:|:---:|:---:|------|
| 시민 좀비 | 3 | 5 | 1 | 기본 잡몹 |
| 육상 좀비 | 2 | 4 | 3 | 빠름, 야간 개인 피해 부여 |
| 강화 좀비 | 10 | 10 | 1 | 높은 공격력 |
| 골렘 좀비 | 14 | 22 | 1 | 고체력 고공격, 후반 등장 |
| 보스 감염체 | 22 | 60 | 1 | 5의 배수 웨이브에 추가 등장 |

### 웨이브 테이블 (이지 모드, 10단계)

| 웨이브 | 구성 | 총 압박 합산 |
|:---:|------|:---:|
| 1 | 시민 좀비 x2 | 6 |
| 2 | 시민 좀비 x2, 육상 좀비 x1 | 8 |
| 3 | 육상 좀비 x2, 시민 좀비 x2 | 10 |
| 4 | 육상 좀비 x3, 시민 좀비 x2 | 12 |
| 5 | 강화 좀비 x1, 육상 좀비 x2 | 14 |
| 6 | 강화 좀비 x1, 육상 좀비 x2, 시민 좀비 x2 | 20 |
| 7 | 강화 좀비 x2, 육상 좀비 x1, 시민 좀비 x3 | 31 |
| 8 | 골렘 좀비 x1, 강화 좀비 x1, 육상 좀비 x2 | 28 |
| 9 | 골렘 좀비 x1, 강화 좀비 x2, 시민 좀비 x3 | 43 |
| 10+ | 골렘 좀비 x2, 강화 좀비 x2, 육상 좀비 x2 | 52 |

> 10웨이브 이후는 최대 테이블(10번)을 반복.

### 보스 웨이브

5의 배수 웨이브(5, 10, 15…)에서 일반 웨이브 공격 **이후** 보스 감염체 추가 타격 발동.
보스 압박 = max(2, ceil((22 - teamArmor) / 4))

### 기지 피해 계산

압박 = max(1, ceil((zombie.pressure - teamArmor - weapon*3) / 6))
기지 HP -= 압박

### 개인 피해

육상 좀비 포함 웨이브에서 생존 플레이어 개별 HP 감소.
군인 `정신력` 적용 시 피해 60%만 적용.

## 기술 메모

### 빌드 방법

```bash
# 개발 서버
npm run dev  # localhost:4173

# 번들 파일 재생성 (Python 필요)
python3 scripts/build_bundle.py
# → zombie-wave-defense-bundle.html 생성
```

### 테스트 배포 (Netlify Drop)

1. app.netlify.com/drop 접속
2. zombie-wave-defense-bundle.html 드래그앤드롭
3. 생성된 URL로 아이폰 Safari 테스트
4. 무료 계정 로그인 시 30일 유지

### 주의사항

- game.js에서 직접 파일을 로컬 경로로 참조 (`assets/images/...`) → 번들 빌드 시 base64 인라인 처리 필수
- Korean 파일명은 macOS에서 NFD로 저장됨 → Python 처리 시 `unicodedata.normalize('NFC', path)` 필요
- `[` `]` 가 포함된 디렉토리명은 bash glob에서 특수문자로 해석됨 → Python `os.listdir()` 사용 권장
