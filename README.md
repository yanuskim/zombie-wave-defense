# 좀비 웨이브 디펜스

브라우저에서 테스트 구동할 수 있는 웹앱 프로토타입입니다.

## 실행

```bash
npm run dev
```

실행 후 브라우저에서 표시되는 로컬 주소로 접속합니다. 기본 주소는 다음과 같습니다.

```text
http://localhost:4173/
```

## 포함된 기능

- 솔로/팀 플레이 선택
- 시작 전 로비와 시작 후 작전 화면 분리
- 오전 7시 토의 페이즈와 오전 9시 작전 시작
- 로컬 핫시트 방식 팀 턴 진행
- 직업 카드 강화
- 기지 방어, 자원 관리, 연구, 조사, 탐험
- 상호작용 행동, 탈수/배고픔 게이지, 500칸 인벤토리
- 제작 도구 데이터와 유물 4개 선택 규칙
- 건물별 랜덤 이벤트
- 중앙 기지형 월드맵과 잠금 숲 지역
- 이지 모드 좀비 도감과 2일차 시작 웨이브 표
- 서사 도감과 엔딩 분기
- 로비 기능 패널, 직업/좀비 도감, 업적, 랭킹, 퀘스트, 출석, 기지 정보 화면
- 웹앱 매니페스트와 서비스 워커

## 검사

```bash
npm run check
```

## 2026-06-11 안정화 패치 (Phase 1)

- 턴 종료 시 식량/물을 먹고 탈수 +40, 배고픔 +30 회복 (탈수 즉사 버그 수정)
- 행동 버튼에 "상호작용" 추가 — 나무/돌/흙 채집으로 제작 레시피 사용 가능
- 기지 체력 상한을 8로 통일
- 게임 중 카드 강화 시 체력/행동력이 풀충전되던 치트 수정
- 쓰러진(체력 0) 플레이어는 행동/턴 전환 불가, 파티 목록에 사망 표시
- 코인/보석/카드/강화/업적/유물이 localStorage에 자동 저장 (새로고침해도 유지)
